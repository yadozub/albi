## Конвенции

- **Транскрипция** — с сохранением орфографии оригинала: тиронские знаки (`⁹`=`-us`), макроны (=опущенный `n`/`m`), лигатуры `æ`/`œ`, длинная `ſ` передаётся как `s`, `u/v` по оригиналу.
- **Перевод** — параллельный (латынь ‖ русский), с сохранением барочного регистра Камюза; пояснения — в сносках под каждой страницей.
- **Разметка достоверности** в аппарате: ✅ документировано · ◐ вероятно · ⚠ под вопросом.
- **Реалии сверяются** по внешним источникам и критическим изданиям; расхождения издания Камюза 1615 с оригиналами отмечаются отдельно.

## Метод чтения оригинала
Страница `albi.pdf` рендерится в PNG (PyMuPDF, 300 dpi) и читается **визуально** — транскрипция идёт с изображения, а не с грязного OCR. Рецепт и карта PDF-страниц — в [HANDOFF.md](HANDOFF.md).
